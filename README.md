# Java-Data-Structures-and-SocialNetwork
## Overview
This project is a Social Network application built in Java as part of an Algorithms and Data Structures course.

The goal of the project was to implement core data structures from scratch (without using Java built-in collections) and apply them in a real-world scenario.

---

## Features
- Create user profiles and corporate profiles
- Add friends between users
- Follow companies
- Post status updates and advertisements
- View user walls (with mixed posts and ads)
- Privacy control for posts (public, friends, private)
- Compute distance between users (graph traversal)
- Find and print connection paths between users

---

## Data Structures Implemented
All data structures were implemented manually:
- Vector (dynamic array)
- Linked List
- Stack
- Queue
- Priority Queue
- Dictionary (key-value structure)
- Circular Vector

---

## How to Run
1. Open the project in IntelliJ (or any Java IDE)
2. Run `Main.java`
3. The console will display:
    - User interactions
    - Walls
    - Friend lists
    - Paths and distances

---

## Purpose
This project demonstrates understanding of:
- Data structure implementation

- Applying theory to a practical system
